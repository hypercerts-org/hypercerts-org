# Architecture

## Environments

Deployment and configuration sets are managed per environment. Tooling like our SDK, API, indexing service and app discern between 'production' and 'test' environments. Our goal is to mimick production as closely as possible while using testnets from our supported mainnets or Layer-x chains.
